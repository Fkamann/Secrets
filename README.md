## Getting Started

- Make sure that you have MongoDB installed and running

```bash
mongod
```

- Install all the dependencies:

```bash
npm install
```

- Then run node:

```bash
node app.js --watch or nodemon app.js
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
